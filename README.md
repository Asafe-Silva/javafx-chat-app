# JavaFX Chat Application

Sistema de chat desktop com múltiplos clientes e salas, desenvolvido em JavaFX.

## Requisitos

- Java 17 ou superior
- Maven 3.6+
- JavaFX 21

## Como Executar no IntelliJ IDEA

1. Abra o projeto no IntelliJ IDEA
2. Certifique-se de que o Maven está configurado corretamente
